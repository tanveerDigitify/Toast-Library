# Toast-Library






# Usage

Add it in your root build.gradle at the end of repositories:


	allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
  
  Add the dependency
  
  	dependencies {
	        implementation 'com.github.tanveerDigitify:Toast-Library:0.0.1'
	}
